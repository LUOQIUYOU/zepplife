# zepplife
zepplife更新步数

## 使用
修改 `config.ini` 文件，填写好ZeppLife账号、密码，要刷的步数区间（程序将在区间中取随机值）。
+ `zepplife`：ZeppLife账号、密码，左边账号，右边密码，可以设置多个。最好用邮箱注册的，小米账号不行
+ `step.min_step`：步数区间最小值
+ `step.max_step`：步数区间最大值

执行 `main.py` 文件。
